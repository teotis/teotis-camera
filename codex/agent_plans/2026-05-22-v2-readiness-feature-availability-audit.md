# 并行任务 C：功能可触达与有效可用审查（非多模态）

## 1. 目标

确认“用户能触达的功能”不是假入口：可真正执行并产出有效结果。

## 2. 功能分组

- 基础：拍照、录像、缩略图回看
- 模式：夜景、人像、文档、人文、专业
- 控件：闪光、变焦、网格、比例、定时、手动参数（如适用）

## 3. 检查点

1. 入口是否可见且可点击
2. 点击后状态是否可感知
3. 状态是否影响真实请求/结果（非仅 UI 假状态）
4. 不支持能力是否明确 `unsupported/degraded`
5. 错误/降级是否可被用户理解

## 4. 产出

- `Feature-Availability-Audit.md`
- 功能矩阵（功能 × 可触达 × 可执行 × 可验证结果）
- P0/P1 功能缺陷清单
