# OpenCamera 2.0 Reference Images

This directory stores project-bound visual references for the 2.0 UI/interaction redesign.

Generated files:

- `camera_cockpit_v2_main.png`
- `camera_cockpit_v2_recording.png`
- `tone_lab_v2_panel.png`
- `lens_lab_v2_panel.png`
- `pro_controls_v2_panel.png`

These images are deterministic PNG mockups generated by `generate_reference_images.py`. They are intended to preserve layout, hierarchy, Chinese labels, panel density, and control grouping for human review and future multimodal review.

They are not a replacement for the Markdown specs in `codex/v2_ui/`. If an image and a written spec disagree, follow the written spec.

To regenerate:

```bash
rtk /Users/dingren/.cache/codex-runtimes/codex-primary-runtime/dependencies/python/bin/python3 codex/v2_ui/reference_images/generate_reference_images.py
```
