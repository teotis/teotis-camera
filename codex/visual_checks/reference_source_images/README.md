# Reference Source Images

Reusable high-quality source images for visual checks, prototype scripts, effect tuning, and manual QA comparisons.

These files are not Android runtime resources. Keep them under `codex/visual_checks/` so tests and local review scripts can reference stable repo paths without shipping them in the app.

## Files

- `daylight-street-portrait-cat-reference.png`: daylight portrait with street depth, bright skin tones, white hair detail, and an animal foreground.
- `rainy-dusk-guitar-portrait-reference.png`: square rainy dusk portrait with wet-road highlights, instrument detail, mixed warm/cool lighting, and shallow background depth.
- `night-city-wall-bike-portrait-reference.png`: night city-wall portrait with warm architectural lights, blue-hour sky, bicycle foreground geometry, and face-detail retention.
