# Status Directory

This directory contains orchestration status files.

- `state.tsv` - Machine-readable state ledger (source of truth)
- `events.jsonl` - Append-only audit log
- `<package-id>.md` - Human-readable package status
- `report.html` - Final HTML report (generated by 07-synthesis-html-report)

Do not edit `state.tsv` manually. Use `orchestrate.sh mark-state` to update it.
