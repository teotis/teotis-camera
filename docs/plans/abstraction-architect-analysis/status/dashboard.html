Wave 1 四个包并行执行，完成后自动触发 Wave 2，依此类推。每个 agent 完成后调用 advance 推进。

项目模块覆盖
  • core/session — Session Kernel（状态机、恢复、诊断）
  • core/device — 设备适配器契约
  • core/media — 媒体管线契约
  • core/mode — 模式目录和插件契约
  • core/settings — 持久化设置
  • core/capability — 能力图
  • core/effect — 效果管线
  • feature/mode-* — 7 个模式插件
  • app — Android/CameraX 集成

操作命令

查看状态
bash docs/plans/abstraction-architect-analysis/launchers/orchestrate.sh status
手动推进（agent 无法运行 shell 时）
bash docs/plans/abstraction-architect-analysis/launchers/orchestrate.sh advance
重试失败的包
bash docs/plans/abstraction-architect-analysis/launchers/orchestrate.sh retry <package-id>
查看 agent 日志
claude logs <session-id> # 例如: claude logs 8cf70cfb
停止 agent
claude stop <session-id>
环境检查
bash docs/plans/abstraction-architect-analysis/launchers/orchestrate.sh doctor --environment
最终报告位置
docs/plans/abstraction-architect-analysis/report.html
