## 视觉分析维度

### 1. 颜色系统分析
- **主色调**：主要颜色、辅助颜色、强调颜色
- **颜色搭配**：颜色组合、颜色对比、颜色和谐
- **颜色含义**：功能颜色、状态颜色、语义颜色
- **颜色一致性**：颜色使用规范、颜色变化规律

### 2. 字体排版分析
- **字体选择**：字体类型、字体家族、字体风格
- **字体大小**：标题大小、正文大小、注释大小
- **行高间距**：行高、段间距、字间距
- **字重粗细**：粗体、常规、细体
- **文字对齐**：左对齐、右对齐、居中对齐

### 3. 间距系统分析
- **元素间距**：按钮间距、面板间距、卡片间距
- **边距系统**：页面边距、组件边距、内容边距
- **内边距**：按钮内边距、输入框内边距、面板内边距
- **间距规律**：间距倍数、间距系统、间距一致性

### 4. 视觉层次分析
- **信息层次**：标题、副标题、正文、注释
- **功能层次**：主要功能、次要功能、辅助功能
- **视觉权重**：颜色权重、大小权重、位置权重
- **对比度**：颜色对比度、大小对比度、位置对比度

### 5. 图像效果分析
- **水印效果**：水印位置、水印大小、水印透明度
- **边框效果**：边框样式、边框颜色、边框宽度
- **阴影效果**：阴影颜色、阴影大小、阴影模糊
- **渐变效果**：渐变方向、渐变颜色、渐变强度

## 分析方法

### 1. 颜色提取
从截图中提取主要颜色，分析颜色系统。

### 2. 尺寸估算
估算界面元素的尺寸、间距、边距等参数。

### 3. 对比度分析
分析颜色对比度，评估可读性和可访问性。

### 4. 美学评估
从用户体验角度评估视觉效果和美学设计。
