## 一致性分析维度

### 1. 视觉风格一致性
- **颜色一致性**：颜色使用、颜色搭配、颜色变化
- **字体一致性**：字体选择、字体大小、字体粗细
- **间距一致性**：元素间距、边距、内边距
- **图标一致性**：图标风格、图标大小、图标颜色
- **阴影一致性**：阴影效果、阴影大小、阴影颜色

### 2. 组件设计一致性
- **按钮一致性**：按钮样式、按钮大小、按钮颜色
- **滑块一致性**：滑块样式、滑块大小、滑块颜色
- **菜单一致性**：菜单样式、菜单结构、菜单交互
- **面板一致性**：面板样式、面板大小、面板背景
- **卡片一致性**：卡片样式、卡片大小、卡片边框

### 3. 交互模式一致性
- **点击一致性**：点击反馈、点击效果、点击流程
- **滑动一致性**：滑动方向、滑动效果、滑动反馈
- **拖拽一致性**：拖拽方式、拖拽反馈、放置效果
- **长按一致性**：长按触发、长按效果、长按菜单

### 4. 信息架构一致性
- **导航一致性**：导航结构、导航方式、导航反馈
- **布局一致性**：布局结构、布局方式、布局规律
- **层次一致性**：信息层次、功能层次、视觉层次
- **流程一致性**：操作流程、反馈流程、完成流程

### 5. 语言文案一致性
- **文案风格**：文案语气、文案长度、文案格式
- **术语一致性**：功能术语、状态术语、操作术语
- **标点符号**：标点使用、标点风格、标点规范
- **大小写**：大小写规则、大小写一致性

## 分析方法

### 1. 横向对比
对比不同截图中的相同元素，识别一致性。

### 2. 纵向对比
对比同一截图中的不同元素，识别一致性。

### 3. 模式识别
识别重复出现的设计模式和交互模式。

### 4. 差异分析
识别不一致的设计和交互，分析原因和影响。

## 产出要求

### 报告结构
1. **摘要**：总体发现和关键问题
2. **视觉风格一致性分析**：颜色、字体、间距、图标一致性分析
3. **组件设计一致性分析**：按钮、滑块、菜单、面板一致性分析
4. **交互模式一致性分析**：点击、滑动、拖拽、长按一致性分析
5. **信息架构一致性分析**：导航、布局、层次、流程一致性分析
6. **语言文案一致性分析**：文案、术语、标点、大小写一致性分析
7. **不一致问题清单**：识别的所有不一致问题
8. **优化建议**：具体的优化点和实施方向

### 优化建议维度
- **视觉统一**：统一颜色、字体、间距、图标设计
- **组件统一**：统一按钮、滑块、菜单、面板设计
- **交互统一**：统一点击、滑动、拖拽、长按交互
- **架构统一**：统一导航、布局、层次、流程设计
- **文案统一**：统一文案风格、术语、标点、大小写
